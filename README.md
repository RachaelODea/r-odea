# r-odea

## Fun Sticker Pack

A small collection of cheerful, hand-crafted SVG stickers — each with the
classic die-cut white border, soft drop shadow, and a friendly face.

### Preview

Open [`index.html`](index.html) in any browser to see the whole pack laid out
in a hoverable gallery.

### Using them

The stickers are plain SVG, so you can:

- Drop them into a webpage with `<img src="stickers/sun.svg">`
- Open them in any vector editor (Figma, Illustrator, Inkscape) to recolor or tweak
- Export them to PNG at any size without losing quality

Have fun!
